Map Helper

The Map Helper file contains functions that assist in working with Maps.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('Map');

The following functions are available:

point_distance()

Returns the distance between two coordinates, in the given unit.

Units can be kilometers (K), nautical miles (N) or by default, miles.

Parameters: latitude (first_point), longitude (first_point), latitude (second_point), longitude(second_point), unit. Example:

echo $point_distance(51.902554,-8.487682, 40.735812,-74.006882, "K");

// returns 5001.9361963688;

coordinates_from_address()

Reverse geocoding of a given address, with cache of previous searchs.

First parameter is the address to search, the second the expected return format. Accepted values: array or text. Example:

echo coordinates_from_address("Cork, Ireland", "text");

// returns 51.8968917, -8.4863157;
