Path Helper

The Path Helper file contains functions that permits you to work with file paths on the server.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('path');

The following functions are available:

my_absolute_path()

This is a complex function that will generate an absolute path from some parameters. It has advanced features like the automatic creation of subfolders.

First parameter:

The path to the file. It can be an array of segments or a raw path.

For example, both things would be valid:

- array("uploads", "temp", "my_file.jpg");

- "uploads/temp/my_file.jpg"

The first version is the recommended way because it's more reliable. The segments will be joined using the desired directory separator.

Second parameter:

By default it will use Codeiginter's installation folder. You can force it to use other folder.

Third parameter:

By default, system's default directory separator. You can force it to use another separator. If you are using Windows it will be"\" by default. On other systems it will use "/" as default.

Fourth parameter:

By default FALSE. Create subfolders, yes/no. It can attempt to create the subfolders of the path if they don't exist in the system.

Fifith parameter:

By default "file". If you set it as "folder", it will be considered as a folder. This is useful if you want to create the subfolders, if it's a file, the last segment of the path will be ignored, if it's a folder, all the segments will be used.

$path = my_absolute_path(array("uploads", "image.jpg"));
//return something like C:\xampp\htdocs\codeigniter\uploads\image.jpg

$path = my_absolute_path("images/image.jpg", "C:\");
// returns C:\images\image.jpg

$path = my_absolute_path(array("uploads", "image.jpg"), "/home/", "/");
// returns /home/uploads/image.jpg

$path = my_absolute_path(array("uploads", "temp1", "image.jpg"), NULL, DIRECTORY_SEPARATOR, TRUE);
// returns C:\xampp\htdocs\codeigniter\uploads\temp1\image.jpg and it creates the "uploads" and "temp1" folders if they didn't exist.

$path = my_absolute_path(array("uploads", "temp1", "images"), NULL, DIRECTORY_SEPARATOR, TRUE, "folder");
// returns C:\xampp\htdocs\codeigniter\uploads\temp1\images and it creates the "uploads", "temp1" AND "images" folders if they didn't exist.
