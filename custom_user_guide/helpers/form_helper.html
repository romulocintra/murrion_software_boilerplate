Form Helper

The Form Helper file contains functions that assist in working with forms.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('form');

The following functions are available:

get_month_dropdown()

Creates an array with all months names.

Example:

get_month_dropdown();

// Returns an array
Array
(
    [01] => January
    [02] => February
    [03] => March
    [04] => April
    [05] => May
    [06] => June
    [07] => July
    [08] => August
    [09] => September
    [10] => October
    [11] => November
    [12] => December
)

get_year_dropdown()

Creates an array with the years between start_year and end_year. If not end_year provided, the array will end at system actual year.

Example

get_year_dropdown(1990, 2000);

// Returns an array
Array
(
    [1990] => 1990
    [1991] => 1991
    [1992] => 1992
    [1993] => 1993
    [1994] => 1994
    [1995] => 1995
    [1996] => 1996
    [1997] => 1997
    [1998] => 1998
    [1999] => 1999
    [2000] => 2000
)

get_hour_dropdown()

Create an array with all hours of the day

Example:

get_hour_dropwdown();

// Returns an array
Array
(
    [00] => 00
    [01] => 01
    [02] => 02
    [03] => 03
    [04] => 04
    [05] => 05
    [06] => 06
    [07] => 07
    [08] => 08
    [09] => 09
    [10] => 10
    [11] => 11
    [12] => 12
    [13] => 13
    [14] => 14
    [15] => 15
    [16] => 16
    [17] => 17
    [18] => 18
    [19] => 19
    [20] => 20
    [21] => 21
    [22] => 22
    [23] => 23
)

get_minute_dropdown()

Create an array with an interval of 60 minutes. You can also provide an interval.

Example:

get_minute_dropdown(15);

// Returns an array with an interval of 15 minutes
Array
(
    [00] => 00
    [15] => 15
    [30] => 30
    [45] => 45
)
