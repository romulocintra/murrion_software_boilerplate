String Helper

The String Helper file contains functions that assist in working with strings.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('string');

The following functions are available:

values_separated_by_comma()

Returns provided arguments separated by comma. Similar to implode, but removing empty values.

Usage example:

$address = array("Country" => "Ireland", "County" => "", "City" => "Cork");
echo values_separated_by_comma($address);

//output: Ireland, Cork

br2nl()

Convert BR tags to new lines. Useful when using a visual HTML editor.

Usage example:

$text = "Dear mum:<br/>Hope you are OK.<br/>Kind regards";
echo br2nl(br2nl);

//output: Dear mum/nHope you are OK./nKind regards

template_replace()

Equivalent to str_replace, for associative arrays, 'key' is replaced with 'value'.

$values = array("January" => 1, "February" => 2, "March" => 3);
$text = "My favourite months are January, February and March";
echo template_replace($values, $text);

//output My favourite months are 1, 2 and 3

Note: To use multiple separate calls to this function simply call the function with no arguments to re-initialize.
