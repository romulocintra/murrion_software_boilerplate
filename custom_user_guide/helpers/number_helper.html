Number Helper

The Number Helper file contains functions that help you work with numeric data.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('my_number');

The following functions are available:

divide_safe()

Divide numbers if the second parameter is not zero, in this case, just returns 0.

echo divide_safe(10, 2); // Returns 5
echo divide_safe(10,0); // Returns 0

my_format_number()

Returns a number in a specific format, including money symbols. Examples:

echo my_format_number(456); // Returns 456
echo my_format_number(456, 2); // Returns 456
echo my_format_number(456.1, 2); // Returns 456.10
echo my_format_number(456.251, 2, "euro"); // Returns €456.25

to_boolean()

Convert to boolean a variable, potentially in text format. Examples:

echo to_boolean (TRUE); //Returns true
echo to_boolean("false"); //Returns false
echo to_boolean (0); //Returns false
